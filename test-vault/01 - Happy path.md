# Happy path

Well-formed tasks, one per expected column. Verify each lands in the right place
and the count badges read: Todo 2, In Progress 2, Done 1, Cancelled 1.

- [ ] Plain todo task
- [?] Custom TODO status (question mark) — should sit in **Todo**
- [/] In-progress task
- [A] Custom IN_PROGRESS status (A) — should sit in **In Progress**
- [x] Completed task ✅ 2026-06-10
- [-] Cancelled task

## With common metadata

- [ ] Buy groceries 🔼 📅 2026-06-15 #errands
- [/] Write quarterly report ⏫ 📅 2026-06-12 #work #report
- [x] Ship release 🔽 ✅ 2026-06-09 #work
